## 問題の特定

スクリーンショットから、以下の問題が確認されました：

### 問題1: 「選択中: worldwide」の表示
- ドイツ語、韓国語、スペイン語、フランス語、繁体字中国語の画面で

## トラブルシューティング

### 問題: まだ「worldwide」と英語で表示される

**原因:** GUIコード内で`"worldwide"`という文字列がハードコードされている可能性

**解決方法:**
1. `gui.py`を開く
2. `"worldwide"`を検索
3. `t("region_worldwide")`または`t_region("worldwide")`に置き換える

例：
```python
# 修正前
selected_text = f"{t('region_selected')} worldwide"

# 修正後
selected_text = f"{t('region_selected')} {t('region_worldwide')}"
```

### 問題: 言語切り替え後も表示が変わらない

**原因:** アプリの再起動が必要な可能性

**解決方法:**
1. アプリを完全に終了
2. アプリを再起動
3. Settings → Language から言語を再選択
