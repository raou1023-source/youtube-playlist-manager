\# YouTube Playlist Manager



年代別・ジャンル別・地域別でYouTube再生リストを自動作成するツール



\## 機能



\- 📅 年代別検索（1960年代～2020年代）

\- 🎵 キーワード別検索

\- 🌍 地域別検索

\- ✅ 公式チャンネル優先

\- 🔄 自動アップデート



\## ダウンロード



\[Releases](https://github.com/raou1023-source/youtube-playlist-manager/releases)から最新版をダウンロード
